Projects & Systems Built

Cloud Detection · Geostationary

Global 6-Satellite Cloud Detection System

Automated cloud detection pipeline ingesting live data from 6 geostationary satellites (GOES-18/19, Himawari-8/9, MSG-2, MTG-0) covering the entire Earth disk. Generates weighted cloud statistics and GeoJSON exports for global weather analysis. Resolved critical MSG-2 GRIB2 coordinate failure by implementing EUMETSAT inverse projection formulas, restoring Indian Ocean coverage at sub-kilometer accuracy.

6 satellites Full Earth coverage 74.6% on-Earth coverage Sub-km accuracy

Python · Satpy · GDAL · EUMETSAT APIs · GeoJSON · Concurrent Processing

Atmospheric Science · ML

Methane Plume Detection & Quantification

CSDNet-based cloud segmentation pipeline for multispectral satellite imagery reducing false positives in methane plume detection. Developed physics-based OpenCV feature extraction workflow for automated plume segmentation from WRF concentration fields. Validated two independent quantification methodologies (IME, cross-sectional, Gaussian) against simulated point-source ground truth.

98% precision 23% fewer false positives 30% faster processing 25% pipeline efficiency gain

Python · OpenCV · WRF-ARW · CSDNet · Scikit-learn · HDF5

98% pipeline speedup 10+ countries 80% 4-week forecast validity 70% reliability gain
