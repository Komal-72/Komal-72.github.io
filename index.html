Cloud Detection System via Geostationary Satellites

Automated cloud detection pipeline ingesting live data from 8 geostationary satellites (GOES-16,18,19, Himawari-8,9, MSG-2,3, MTG-0) covering the entire Earth disk, with rigorous sub-pixel fractional coverage estimation using Shapely polygon intersection. Implemented ground-up SEVIRI coordinate reconstruction from raw GRIB navigation keys using WMO GDT 3.90 inverse projection, handling rarely-documented scanning mode differences between EUMETSAT prime and IODC services.
